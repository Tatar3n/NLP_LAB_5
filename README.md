# NLP_LAB_5
Telegram Бот для лабораторной NLP.
Этот код представляет собой Telegram бота, который взаимодействует с локально запущенной языковой моделью через API.
Основные функции:
1. Команды бота:
- /start - Приветственное сообщение со списком всех доступных команд
- /model - Показывает название языковой модели
- /clear - Очищает историю диалога конкретного пользователя
2. Основная логика работы:
- Бот поддерживает беседу с каждым пользователем отдельно
- Все сообщения пользователя сохраняются в истории диалога
- Ответы модели также сохраняются в истории для поддержания контекста беседы
Чтобы работало:
- Создать внутри проекта файл .env с переменной BOT_TOKEN содержащий токен Telegram бота.
